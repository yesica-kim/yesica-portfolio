# Yesica Portfolio v0.1.0 기능정의서

이 문서는 `docs/현재_기능정의서.md`의 v0.1.0 스냅샷이다.

## 현재 버전

v0.1.0

## 기능 범위

- Next.js App Router 기반 페이지 라우팅
- Tailwind CSS 기반 다크 UI
- Framer Motion 기반 기본 모션
- 프로젝트 데이터 TypeScript 파일 분리
- Vercel 배포 운영 구조

## 페이지 구성

- `/`
- `/about`
- `/projects`
- `/projects/[slug]`
- `/contact`
- `/writing`

## 프로젝트 상세 페이지

구성:

- Project Hero
- Overview
- Problem
- Goal
- Process
- Design Solution
- Result / Learning

## 외부 연동

- GitHub: `yesica-kim/yesica-portfolio`
- Production: `yesica-portfolio`
- Development: `yesica-portfolio-dev`

## 검증 기준

- `npm run lint`
- `npm run build`
- Vercel dev/prod 배포 확인
