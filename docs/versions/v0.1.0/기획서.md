# Yesica Portfolio v0.1.0 기획서

이 문서는 `docs/현재_기획서.md`의 v0.1.0 스냅샷이다.

## 프로젝트 개요

Yesica Portfolio는 13년차 UI/UX 디자이너 Yesica의 작업과 방향성을 보여주는 개인 브랜드 포트폴리오 사이트다.

## 현재 목적

- 포트폴리오 기본 구조 구축
- 실제 서비스 경험, 개인 제품 제작, AI 협업 메이킹 방향성 표현
- 추후 프로젝트 콘텐츠를 쉽게 채울 수 있는 틀 마련
- Vercel 기반 실서버/개발서버 운영 구조 마련

## 핵심 사용자

- 채용 담당자
- 협업 파트너
- 프리랜서 클라이언트
- 제품 디자인과 AI 협업 메이킹에 관심 있는 사람

## 핵심 가치

- Product Designer
- UI/UX Designer
- Independent Maker
- AI Collaboration
- Real Product Builder
- Life System Designer

## 현재 주요 기능

- Home Hero
- About
- Projects
- Project Detail
- Contact
- Writing 기본 페이지
- 프로젝트 데이터 분리
- 다크 테마 UI
- 반응형 레이아웃
- 개발/실서버 배포 구조
