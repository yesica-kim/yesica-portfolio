# Yesica Portfolio 현재 기획서

## 프로젝트 개요

Yesica Portfolio는 13년차 UI/UX 디자이너 Yesica의 작업과 방향성을 보여주는 개인 브랜드 포트폴리오 사이트다. 단순 취업용 포트폴리오가 아니라 실제 서비스 경험, 개인 제품 제작, AI 협업 메이킹, 프로덕트 사고를 함께 보여주는 입구로 운영한다.

## 현재 목적

- 포트폴리오 사이트의 기본 구조를 빠르게 구축한다.
- 추후 프로젝트별 상세 사례와 실제 링크를 쉽게 채워 넣을 수 있는 틀을 마련한다.
- 프리랜서, 취업, 협업, 창업, 개인 브랜드 확장에 함께 사용할 수 있는 기반을 만든다.
- 여러 디바이스에서 확인 가능한 개발/실서비스 배포 흐름을 갖춘다.

## 핵심 사용자

- Yesica의 작업을 검토하는 채용 담당자 또는 협업 파트너
- 프리랜서 디자인 작업을 의뢰하려는 클라이언트
- 실제 서비스와 개인 프로젝트 경험을 함께 보고 싶은 동료 디자이너/메이커
- AI 협업 기반 제품 제작 방향에 관심 있는 사람

## 핵심 가치

- 실제 제품과 운영 구조를 이해하는 Product Designer
- 복잡한 사용자 흐름과 관리자 시스템을 설계할 수 있는 UI/UX Designer
- 개인 문제를 제품으로 만들고 배포까지 진행하는 Independent Maker
- AI와 함께 제품을 빠르게 실험하고 구현하는 협업형 메이커

## 현재 주요 기능

- 다크 테마 기반 Home Hero
- About, Projects, Project Detail, Contact, Writing 페이지 라우팅
- 프로젝트 데이터 분리 관리
- 프로젝트 상세 공통 템플릿
- 반응형 레이아웃
- Framer Motion 기반 가벼운 진입/hover 모션
- Vercel 실서버/개발서버 운영 구조
- Schedule Block 공식 소개/정책/지원 페이지를 포트폴리오 하위 경로로 제공

## 현재 버전/상태

- 현재 기준 버전: v0.1.0
- 상태: 기본 사이트 골격 구축 완료, 콘텐츠 확장 전 단계
- 배포:
  - Production: `main` 브랜치 기준
  - Development: `dev` 브랜치 기준

## 운영 방식

- 작은 문구 수정과 UI 조정은 프로젝트 채팅에서 바로 진행한다.
- 작업 중 확인은 `dev` 브랜치와 개발 서버 URL을 기준으로 한다.
- 공개 가능한 안정 버전은 `main` 브랜치에 반영한다.
- 추적 가치가 있는 릴리즈 단위 변경은 `docs/변경로그.md`에 기록한다.

## 앞으로의 방향

- 실제 프로젝트별 케이스 스터디 콘텐츠 채우기
- Contact 페이지의 실제 이메일, 브런치, GitHub, LinkedIn 링크 연결
- Brunch/Writing 영역 확장
- 필요 시 Lab, Studio, Resume 페이지 추가
- 실제 제품 링크와 App Store/Live Product 링크 연결
- Schedule Block 앱 심사/공식 문서 요구사항에 맞춘 정책 페이지 유지
